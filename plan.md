# Party Pals Arena — Roadmap to a Fun, Marketable Party Game

## Current Situation

The project already contains:

* 30 playable mini-games
* Working match flow
* Local multiplayer
* 3D arena framework
* Shared camera system
* Mobile + PC support

The project is no longer in the "build more content" phase.

The project is now in the "find the fun and polish it" phase.

---

# Phase 1 — Find the Top Games

## Goal

Identify the 5–10 mini-games that consistently create:

* laughter
* tension
* shouting
* rematches
* memorable moments

These games become the core experience.

---

## Create a Playtest Scorecard

For every mini-game rate:

### Fun

1–5

Did players enjoy it?

### Clarity

1–5

Did players immediately understand what to do?

### Excitement

1–5

Did anything dramatic happen?

### Replayability

1–5

Would players want another round?

### Spectator Value

1–5

Is it fun to watch?

### Total

Maximum score = 25

---

## Classification

### S Tier (22–25)

Must keep.

Polish heavily.

### A Tier (18–21)

Keep.

Improve where needed.

### B Tier (14–17)

Optional.

Polish only after S/A games.

### C Tier (<14)

Candidate for removal or redesign.

---

# Phase 2 — Build the Core Roster

## Goal

Reduce focus from 30 games to a core set.

Target:

5–10 flagship mini-games.

Example categories:

### Survival

* Last player alive
* Dodge hazards

### Combat

* Tank battle
* Arena push

### Racing

* Sprint
* Obstacle race

### Territory

* King of the hill
* Capture zones

### Chaos

* Hot potato
* Bomb pass

---

## Rule

Every category must contain at least one standout game.

Avoid having:

* 5 similar tank games
* 4 similar dodge games
* 3 similar snake games

Variety is more valuable than quantity.

---

# Phase 3 — Establish Visual Identity

## Goal

Make screenshots instantly recognizable.

Current issue:

Prototype visuals.

Target:

Colorful toy-box party game.

---

## Environment

Replace:

* Dark walls
* Dark floors
* Empty arenas

With:

* Chunky toy walls
* Colorful floor tiles
* Decorative props
* Bright lighting

---

## Characters

Upgrade mascots:

* Large eyes
* Rounded silhouettes
* Idle animation
* Victory animation

Keep geometry simple.

Focus on personality.

---

## UI

Upgrade:

* Scores
* Timers
* Round announcements
* Victory screens

Every UI element should feel celebratory.

---

# Phase 4 — Add Game Feel

## Goal

Increase perceived quality without changing gameplay.

---

## Movement

Add:

* Bounce
* Dust particles
* Small squash/stretch

---

## Hits

Add:

* Impact flash
* Particles
* Tiny camera shake

---

## Wins

Add:

* Confetti
* Crown
* Celebration animation
* Score pop

---

## Objective

Every action should feel satisfying.

---

# Phase 5 — Improve Match Flow

## Goal

Make entire matches exciting.

---

## Match Structure

Current:

First to 5 points.

Keep this.

---

## Add

### Match Point

Announce when a player is one point from victory.

### Comeback Moments

Highlight close games.

### MVP Screens

Show fun statistics after matches.

---

# Phase 6 — Playtesting

## Rule

Never rely on personal opinion alone.

Watch real players.

Do not explain.

Observe.

---

## Track

For each mini-game:

### Laughter

0–5

### Confusion

0–5

### Rematch Requests

0–5

### Engagement

0–5

---

## Warning Signs

Players:

* stop talking
* check phones
* look bored
* ask when the round ends

These games need redesign.

---

# Phase 7 — Cut Ruthlessly

## Important

Do not become attached to mini-games.

A game being finished does not mean it should stay.

If a mini-game consistently scores poorly:

* redesign it
* merge it
* remove it

The goal is a stronger overall game.

---

# Phase 8 — Content Lock

## After Core Polish

When:

* visuals are consistent
* top games are polished
* playtests are positive

Stop adding mini-games.

Focus on:

* balancing
* presentation
* onboarding
* bug fixing

---

# Success Criteria

A successful Party Pals Arena session should produce:

* laughter
* shouting
* rivalry
* "one more match"

Players should remember moments, not features.

The goal is not:

"30 mini-games."

The goal is:

"That game was ridiculously fun."
