Control asset pack for touch UI variants.

Files:
- `tap_anywhere.png`: full-screen tap prompt for reaction or timing games.
- `tap_button.png`: single subtle tap button for simple one-action games.
- `joystick_base.png`: low-noise analog movement ring.
- `joystick_knob.png`: separate analog stick knob so UI can animate offset.
- `swipe_horizontal.png`: lane-switch or dodge-left-right prompt.
- `swipe_vertical.png`: climb, lift, or drop prompt.
- `drag_anywhere.png`: drag gesture hint for aim or steer-anywhere games.
- `hold_button.png`: hold-to-charge or sustain input prompt.
- `left_right_pad.png`: two-zone touch pad for binary movement games.
- `controls_preview.png`: quick visual sheet of the full pack.

Suggested mapping:
- Reaction/timing games: `tap_anywhere` or `tap_button`
- Racing lane games: `left_right_pad` or `swipe_horizontal`
- Free-move action games: `joystick_base` + `joystick_knob`
- Aim/steer direct-manipulation games: `drag_anywhere`
- Charge/jump/boost games: `hold_button`
